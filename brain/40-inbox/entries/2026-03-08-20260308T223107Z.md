# Note

Memory policy update: replaced 'always manually set an alarm' with auto system-alarm model. For non-alarm tasks, prefer --timeout-sec > 20 so WakeBridge creates system-alarm(parent:<task_id>) and auto-cancels on early completion; if missing, emit immediate WB_WARN at start. Manual alarm(tag=alarm) is optional for ad-hoc checks.
